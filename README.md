Project Overview

This project focuses on exploratory data analysis and operational performance evaluation using a supply chain dataset containing sales, inventory, logistics, transportation, and supplier-related metrics.

The analysis was conducted entirely in Python using business-focused analytical workflows to identify operational trends, demand patterns, logistics performance, and supplier efficiency across multiple product categories.

The project combines:

* Data understanding
* Data quality assessment
* Exploratory data analysis (EDA)
* Operational KPI analysis
* Business insight generation
* Data visualization and storytelling

⸻

Tools & Libraries Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

⸻

Business Problems Addressed

This project investigates several operational and business-related supply chain questions:

* Which product categories generate the highest revenue?
* Which customer demographics contribute the most sales?
* Are inventory levels aligned with product demand?
* Which products experience slower replenishment cycles?
* Which shipping carriers are operationally efficient?
* Which transportation modes are more cost-effective?
* Are higher manufacturing costs translating into higher revenue?
* Which suppliers demonstrate stronger operational performance?
